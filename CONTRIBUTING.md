# Contributing

Gracias por tu interés en contribuir al proyecto Solana Million Grid.

Cómo contribuir
- Abre un issue para proponer mejoras o reportar bugs.
- Crea una rama nueva para tu cambio: `git checkout -b feat/nombre-idea`
- Añade tests y documentación si procede.
- Haz un PR contra `main` y describe claramente los cambios.

Normas rápidas
- No subas secretos ni archivos `.env`.
- Sigue el styleguide (ESLint + Prettier).
- Si tocas la persistencia, documenta migraciones.

Gracias :)
